# Industrial Spec Parser

> ⚠️ **Status: Internal tool — planned for public release in Q3 2026**

## Purpose
This tool automates the extraction and validation of technical specifications from industrial equipment suppliers (including Chinese manufacturers). It helps engineering teams:
- Parse heterogeneous PDF/Excel technical documents
- Standardize catalyst and equipment data
- Reduce manual review time and human error

## Planned Features (v1.0)
- [ ] PDF/Excel parser for technical specs
- [ ] Validation rules for catalyst parameters
- [ ] Export to structured JSON/CSV
- [ ] CLI interface for batch processing
- [ ] Documentation in English + Russian

## Tech Stack (planned)
- Python 3.10+
- pandas, lxml, PyPDF2 / pdfplumber
- pytest for test coverage
- Poetry for dependency management

## Why Open Source?
Industrial automation lacks good OSS tools for working with international supplier data. By open-sourcing the core parsing logic, I hope to help other engineers in chemical/process industries save time and reduce errors.

## Current Status
🔒 Core logic is currently internal (company policy). Refactoring for public release is planned for Q3 2026.
